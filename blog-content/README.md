# Blog posts

Add `.md` files here, then run:

```bash
npm run build:blog
```

## Template

```markdown
# Blog Post: Your title

## Meta Information
- **Title:** Your post title
- **Slug:** your-post-slug
- **Excerpt:** Short description for listings
- **Date:** 2026-06-18
- **Tags:** category, keywords

---

## Content (HTML)

\`\`\`html
<h2>Heading</h2>
<p>Your content here. Link to <a href="https://kama.biz/tbilisi">KAMA.BIZ</a>.</p>
\`\`\`
```

After adding a post, add its URL to `sitemap-pages.xml`.
